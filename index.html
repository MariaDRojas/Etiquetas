Listado de 102 Etiquetas HTML

# Etiqueta Descripción Ejemplo Categoría
1 <html> Es la etiqueta raíz del documento. Todo el contenido de la página web debe estar dentro de esta etiqueta. Indica al navegador que el archivo es un documento HTML. <html lang="es"> ... </html> Estructura completa de cualquier página web.
2 <head> Contiene información que no se muestra directamente en la página, como el título, metadatos y enlaces a archivos externos. <head><title>Mi página</title></head> Configurar el título y codificación del sitio.
3 <title> Define el nombre que aparece en la pestaña del navegador. <title>>Tienda Online</title> Mostrar el nombre de una empresa en la pestaña.
4 <body> Contiene todo el contenido visible de la página: textos, imágenes, tablas, formularios, etc. <body><h1>Bienvenidos</h1></body> Mostrar información principal al usuario.
5 <h1> Encabezado principal. Se usa para el título más importante del documento. <h1>>Mi Biografía</h1> Título principal de una página.
6 <h2> Subtítulo de segundo nivel. Organiza el contenido en secciones. <h2>>Datos Personales</h2> Subtítulo dentro de una biografía.
7 <h3> Encabezado de tercer nivel, usado para subdividir secciones. <h3>>Experiencia Laboral</h3> Subcategorías dentro de un tema.
8 <p> Define un párrafo de texto. Se usa para escribir contenido informativo. <p>Soy estudiante de informática.</p> Describir información personal.
9 <br> Inserta un salto de línea sin crear un nuevo párrafo. Línea 1<br>Línea 2 Separar dirección o poemas.
10 <hr> Inserta una línea horizontal para separar contenido. <hr> Separar secciones en un documento.
11 <a> Crea enlaces que permiten navegar a otras páginas o sitios web. <a href="https://google.com">Ir a Google</a> Botón para ir a otra página.
12 <img> Permite insertar imágenes en la página. <img src="foto.jpg" alt="Mi foto"> Mostrar una foto de perfil.
13 <ul> Crea una lista desordenada (con viñetas). <ul><li>Manzana</li></ul> Lista de compras.
14 <ol> Crea una lista ordenada (numerada). <ol><li>Paso 1</li></ol> Pasos de un proceso.
15 <li> Elemento individual dentro de una lista. <li>Elemento</li> Ítem dentro de menú.
16 <table> Define una tabla para organizar datos en filas y columnas. <table></table> Tabla de calificaciones.
17 <tr> Define una fila dentro de una tabla. <tr></tr> Fila con datos de un estudiante.
18 <td> Celda de datos dentro de una tabla. <td>90</td> Nota de una materia.
19 <th> Celda de encabezado en una tabla. <th>Nombre</th> Encabezado de columna.
20 <form> Define un formulario para enviar información del usuario. <form action="guardar.php"></form> Formulario de registro.
21 <input> Permite crear campos interactivos dentro de un formulario. Puede ser de distintos tipos como texto, contraseña, correo, número, fecha, etc. <input type="text" placeholder="Escribe tu nombre"> Campo para que el usuario escriba su nombre.
22 <button> Crea un botón clickeable que puede enviar formularios o ejecutar acciones. <button type="submit">Enviar</button> Botón para enviar un formulario.
23 <label> Define una etiqueta descriptiva para un campo de formulario, mejorando accesibilidad. <label for="correo">Correo:</label> Identificar claramente un campo de entrada.
24 <select> Crea una lista desplegable para seleccionar una opción. <select><option>Opción 1</option></select> Seleccionar país en un registro.
25 <option> Define una opción dentro de una lista desplegable. <option value="rd">República Dominicana</option> Elegir una ciudad o país.
26 <textarea> Permite ingresar texto largo en múltiples líneas. <textarea rows="4" cols="30"></textarea> Campo para comentarios.
27 <div> Contenedor genérico usado para agrupar elementos y organizar la estructura. <div>Contenido agrupado</div> Agrupar una sección completa.
28 <span> Contenedor en línea utilizado para aplicar estilos o modificar parte de un texto. <span>Texto resaltado</span> Resaltar una palabra dentro de un párrafo.
29 <strong> Indica que el texto es importante. Generalmente se muestra en negrita. <strong>Advertencia</strong> Resaltar información importante.
30 <em> Da énfasis al texto. Normalmente se muestra en cursiva. <em>Importante</em> Destacar una palabra clave.
31 <b> Aplica negrita sin significado semántico especial. <b>Texto en negrita</b> Destacar visualmente una palabra.
32 <i> Muestra texto en cursiva sin significado semántico adicional. <i>Texto en cursiva</i> Títulos de libros o palabras extranjeras.
33 <u> Subraya el texto. <u>Texto subrayado</u> Resaltar información.
34 <small> Muestra texto en un tamaño más pequeño. <small>Nota legal</small> Texto de derechos reservados.
35 <mark> Resalta texto como si estuviera marcado con un resaltador. <mark>Importante</mark> Señalar resultados de búsqueda.
36 <del> Representa texto eliminado o tachado. <del>$100</del> Precio anterior en una tienda.
37 <ins> Representa texto insertado o agregado. <ins>$80</ins> Nuevo precio actualizado.
38 <sub> Muestra texto en subíndice. H<sub>2</sub>O Fórmulas químicas.
39 <sup> Muestra texto en superíndice. 10<sup>2</sup> Potencias matemáticas.
40 <nav> Define una sección de navegación con enlaces. <nav><a href="#">Inicio</a></nav> Menú principal de un sitio web.
41 <header> Representa la parte superior de una página o sección. Generalmente contiene el título, logotipo o menú de navegación. <header><h1>Mi Sitio Web</h1></header> Encabezado principal con logo y título.
42 <footer> Define el pie de página de un documento o sección. Contiene información como autor, derechos reservados o enlaces adicionales. <footer><p>© 2026 Todos los derechos reservados</p></footer> Información de copyright al final de la página.
43 <section> Representa una sección temática dentro del documento. <section><h2>Servicios</h2></section> Sección de servicios en una página empresarial.
44 <article> Define contenido independiente que puede distribuirse por separado. <article><h2>Noticia</h2></article> Publicación de blog o noticia.
45 <aside> Contiene contenido relacionado pero secundario. <aside>Publicidad</aside> Barra lateral con anuncios.
46 <main> Representa el contenido principal del documento. <main><p>Contenido principal</p></main> Parte central de la página.
47 <figure> Se usa para agrupar contenido multimedia como imágenes o gráficos. <figure><img src="foto.jpg"></figure> Imagen con descripción.
48 <figcaption> Proporciona una descripción para un elemento <figure>. <figcaption>Foto del evento</figcaption> Texto explicativo debajo de una imagen.
49 <audio> Permite reproducir archivos de audio. <audio controls src="audio.mp3"></audio> Reproductor de música.
50 <video> Permite reproducir videos en la página web. <video controls src="video.mp4"></video> Video promocional.
51 <source> Especifica múltiples fuentes para audio o video. <source src="video.mp4" type="video/mp4"> Ofrecer diferentes formatos de video.
52 <iframe> Inserta otra página web dentro de la actual. <iframe src="https://example.com"></iframe> Insertar un mapa de Google.
53 <canvas> Permite dibujar gráficos mediante JavaScript. <canvas width="200" height="100"></canvas> Crear gráficos dinámicos.
54 <svg> Define gráficos vectoriales escalables. <svg width="100" height="100"></svg> Crear iconos personalizados.
55 <meta> Define metadatos como codificación y descripción. <meta charset="UTF-8"> Configurar caracteres especiales.
56 <link> Conecta recursos externos como hojas de estilo. <link rel="stylesheet" href="estilo.css"> Vincular archivo CSS.
57 <style> Permite escribir estilos CSS dentro del documento HTML. <style>body{background:gray;}</style> Cambiar color de fondo.
58 <script> Incluye código JavaScript en la página. <script>alert("Hola");</script> Mostrar mensaje emergente.
59 <noscript> Muestra contenido si el navegador no soporta JavaScript. <noscript>Activa JavaScript</noscript> Aviso para usuarios.
60 <details> Crea un elemento desplegable que el usuario puede abrir o cerrar. <details><summary>Ver más</summary>Contenido</details> Preguntas frecuentes.
61 <summary> Define un título visible para el elemento <details>. Al hacer clic, muestra u oculta el contenido adicional. <details> <summary>Más información</summary> Contenido oculto </details> Encabezado de una sección desplegable en preguntas frecuentes.
62 <dialog> Representa una ventana o cuadro de diálogo interactivo. <dialog open>Mensaje importante</dialog> Ventana emergente de confirmación.
63 <progress> Muestra el progreso de una tarea en curso. <progress value="70" max="100"></progress> Barra de carga de un archivo.
64 <meter> Representa una medida dentro de un rango conocido. <meter value="8" min="0" max="10"></meter> Nivel de batería o calificación.
65 <fieldset> Agrupa elementos relacionados dentro de un formulario. <fieldset> <legend>Datos personales</legend> </fieldset> Agrupar información de contacto.
66 <legend> Define un título para un <fieldset>. <legend>Registro</legend> Título de sección en un formulario.
67 <datalist> Proporciona una lista de opciones sugeridas para un campo <input>. <input list="paises"> <datalist id="paises"> <option value="República Dominicana"> </datalist> Sugerencias automáticas al escribir.
68 <optgroup> Agrupa opciones relacionadas dentro de un <select>. <select> <optgroup label="Frutas"> <option>Manzana</option> </optgroup> </select> Categorías dentro de una lista desplegable.
69 <output> Muestra el resultado de un cálculo realizado en un formulario. <output>Resultado</output> Mostrar total calculado automáticamente.
70 <time> Representa fechas y horas en formato legible y estructurado. <time datetime="2026-02-27">27 de febrero de 2026</time> Publicar fecha de una noticia.
71 <code> Muestra fragmentos de código en formato especial. <code>console.log("Hola");</code> Mostrar ejemplo de programación.
72 <pre> Muestra texto respetando espacios y saltos de línea. <pre> Línea 1 Línea 2 </pre> Mostrar código formateado.
73 <blockquote> Define una cita extensa proveniente de otra fuente. <blockquote>La educación es el arma más poderosa.</blockquote> Mostrar una cita famosa.
74 <q> Define una cita corta en línea. <q>El conocimiento es poder</q> Citar una frase breve.
75 <cite> Indica el título de una obra o referencia. <cite>Don Quijote de la Mancha</cite> Referenciar un libro.
76 <abbr> Define una abreviatura con su significado. <abbr title="HyperText Markup Language">HTML</abbr> Mostrar significado al pasar el cursor.
77 <address> Contiene información de contacto del autor o empresa. <address>correo@ejemplo.com</address> Datos de contacto en el pie de página.
78 <bdi> Aísla una parte del texto para manejar correctamente direcciones de escritura. <bdi>Texto</bdi> Mostrar nombres en diferentes idiomas.
79 <bdo> Cambia la dirección del texto (izquierda a derecha o viceversa). <bdo dir="rtl">Texto</bdo> Mostrar texto en árabe.
80 <map> Define un mapa de imagen interactivo. <map name="mapa"></map> Imagen con áreas clickeables.
81 <area> Define un área interactiva dentro de un mapa de imagen (<map>). Permite crear zonas clickeables. <area shape="rect" coords="0,0,100,100" href="pagina.html"> Hacer clic en una parte específica de una imagen.
82 <colgroup> Agrupa columnas dentro de una tabla para aplicarles formato común. <colgroup><col span="2"></colgroup> Aplicar formato a varias columnas.
83 <col> Define propiedades para una columna específica dentro de una tabla. <col> Establecer ancho de columna.
84 <caption> Define el título de una tabla. <caption>Tabla de Calificaciones</caption> Mostrar el nombre descriptivo de una tabla.
85 <tbody> Agrupa el contenido principal de una tabla. <tbody><tr><td>Dato</td></tr></tbody> Contener los datos de estudiantes.
86 <thead> Agrupa el encabezado de una tabla. <thead><tr><th>Nombre</th></tr></thead> Encabezados de columnas.
87 <tfoot> Agrupa el pie de una tabla. <tfoot><tr><td>Total</td></tr></tfoot> Mostrar totales al final de una tabla.
88 <embed> Inserta contenido externo como PDF o multimedia. <embed src="archivo.pdf"> Mostrar un PDF dentro de la página.
89 <object> Incrusta recursos externos como aplicaciones o multimedia. <object data="archivo.pdf"></object> Integrar contenido externo.
90 <param> Define parámetros para un objeto incrustado. <param name="autoplay" value="true"> Configurar opciones de un objeto multimedia.
91 <track> Define subtítulos o descripciones para elementos <video> o <audio>. <track kind="subtitles" src="sub.vtt"> Agregar subtítulos a un video.
92 <wbr> Sugiere un posible punto de salto de línea dentro de una palabra larga. Super<wbr>califragilistico Mejorar visualización en palabras extensas.
93 <data> Asocia contenido visible con un valor legible por máquinas. <data value="12345">Producto A</data> Identificar productos con código interno.
94 <kbd> Representa entrada del usuario desde teclado. <kbd>Ctrl</kbd> + <kbd>C</kbd> Mostrar atajos de teclado.
95 <samp> Representa salida de un programa. <samp>Error 404</samp> Mostrar mensaje de error del sistema.
96 <var> Define una variable en contexto matemático o de programación. <var>x</var> = 10 Representar variable en fórmula.
97 <ruby> Representa anotaciones para idiomas asiáticos. <ruby>漢<rt>kan</rt></ruby> Mostrar pronunciación.
98 <rt> Define la pronunciación dentro de <ruby>. <rt>kan</rt> Texto de pronunciación.
99 <rp> Proporciona paréntesis para navegadores que no soportan <ruby>. <rp>(</rp> Compatibilidad con navegadores antiguos.
100 <menu> Representa una lista de comandos o menú contextual. <menu><li>Opción</li></menu> Menú de opciones.
101 <menuitem> Define un comando dentro de un menú (etiqueta obsoleta en HTML5). <menuitem label="Guardar"> Opción dentro de un menú contextual.
102 <marquee> Etiqueta obsoleta que crea texto en movimiento (no recomendada en HTML moderno). <marquee>Texto en movimiento</marquee> Texto desplazándose horizontalmente.
